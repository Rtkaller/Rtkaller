# Rtkaller
 
`Rtkaller`  is an unsupervised coverage-guided kernel fuzzer for RTOS fuzzing. 

Core features are:
 * Task schema based.
 * Task state awared.
 * Interface awared.
 * Real time code oriented. 

Most of the documentation at this moment is related to the rt-linux kernel.
